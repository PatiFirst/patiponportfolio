Rescape

A VR game with heart rate module to escape the robots
Video Preview

This is a project in "NSC2021". Rescape is a VR-RPG game that made with Unity and play with heart rate module that made with Arduino. I'm responsible to make a heart rate module and design some map in this game

The heart rate module during in this game is show on right hand and depend on a Detect Circle. If detect circle is detect to a robot, the game will restart. So If detect circle is wider, player will escape harder.

Rescape is a game about "Teen" who is growing up with "MOM" that is a robot. One Day, he find "Will" who have injury and talking with him how to escape this place with the robots.
